#People Slides

## Installation

Requirements:

* make
* nodejs && npm
* bower

Then:

```bash
make install
```

## Development

```bash
make serve
```

## Build

```bash
make build
```

## How to....

### How to add a new presentation?

Just add a new directory in `src/slides/` with a `slides.md` inside.
Then add the description of your presentation in `js/slides.js`.

Voilà!
